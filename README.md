## 🚀 Project Overview

The **Interactive Thyroid Disease Prediction System** is a web-based application that analyzes user symptoms and predicts potential thyroid conditions using a **symptom-based scoring model**.

It provides:

* 📊 Probability-based predictions
* ⚠️ Risk level classification
* 📋 Personalized medical recommendations

This tool is designed for **educational and awareness purposes**, helping users understand possible thyroid-related conditions.

## ⚙️ How It Works

1. User inputs health details:

   * Age, Gender
   * Fatigue level
   * Weight changes
   * Temperature sensitivity
   * Mood changes
   * Heart rate
   * Additional symptoms (dry skin, hair loss, etc.)

2. The system processes inputs using a scoring model

3. Calculates probabilities for:

   * Hypothyroidism
   * Hyperthyroidism
   * Normal condition

4. Displays:

   * Prediction result
   * Probability percentage
   * Risk level
   * Medical recommendations

## 🧠 Core Logic

The system uses a **rule-based scoring algorithm** where:

* Each symptom contributes to a score
* Scores are normalized into probabilities
* The highest probability determines the predicted condition
