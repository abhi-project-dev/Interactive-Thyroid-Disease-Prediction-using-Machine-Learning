# Interactive-Thyroid-Disease-Prediction-using-Machine-Learning
The Interactive Thyroid Disease Prediction System is a web-based application designed to provide an intuitive and user-friendly interface for assessing thyroid health risks. The system uses a symptom-driven scoring model to analyze user inputs and generate predictions for possible thyroid conditions.
Users can input various health indicators such as fatigue levels, weight changes, temperature sensitivity, mood variations, heart rate, and other symptoms through an interactive form. Based on these inputs, the system evaluates patterns commonly associated with thyroid disorders and calculates probability scores for:
Hypothyroidism (Underactive Thyroid)
Hyperthyroidism (Overactive Thyroid)
Normal Thyroid Function
The application then displays:
A predicted condition
Probability percentage
Risk level (Low / Medium / High)
Detailed explanation of the condition
Personalized medical recommendations

🔹 How It Works

Users input symptoms using sliders, dropdowns, and checkboxes
The system processes inputs using a scoring logic model
It calculates probabilities for thyroid conditions
Displays results along with actionable insights
